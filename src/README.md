# 📚 디지털 독서 인터랙티브 게임 디자이너

초등학교 2학년부터 중학교 3학년까지 사용할 수 있는 AI 기반 몰입형 인터랙티브 독서게임 생성 시스템

---

## 🎯 프로젝트 개요

- **대상**: 초등학교 2학년 ~ 중학교 3학년
- **형식**: 고전소설 프리퀄 형태의 2인칭 인터랙티브 게임
- **목적**: 작품의 세계관과 인물을 체험하며 독서 흥미 유발
- **완성 작품**: 35개 (초등 2-3학년 16개, 초등 4-6학년 14개, 중학교 1학년 5개)
- **버전**: 1.0.5 (강력한 캐시 무효화 시스템)
- **모범 사례**: 파우스트 (40개 장면, 이미지 37곳 개선, 사람 얼굴 0개)

## 🎨 이미지 교체 작업 프로세스

### 원칙
1. **작품의 시대적 배경이 절대 기준**
   - 페이지 제목 키워드가 아닌 작품 시대를 우선시
   - 조선시대 작품 = 100% 조선시대 이미지
   
2. **현대 요소 제로 톨러런스**
   - 배경에 조금이라도 현대 건물이 보이면 즉시 교체
   - 전봇대, 전깃줄, 아파트 등 절대 금지

3. **자동 문서화**
   - 교체 완료 시 IMAGE_ARCHIVE.md 자동 업데이트
   - 새로운 원칙 발견 시 규칙 문서 자동 반영

### 홍계월전 교체 작업의 의의
**발견**: 페이지 제목의 현대적 키워드("변화", "세대")에 현혹되어 현대 이미지를 선택하는 실수 발견
- ❌ "변화를 위한 용기" → 현대 태권도 대회
- ❌ "다음 세대를 위한 가르침" → 현대 무대

**확립된 원칙**: 작품의 시대적 배경 > 페이지 제목 키워드
- 조선시대 작품은 모든 이미지가 조선시대여야 함
- 이 원칙을 IMAGE_GUIDELINES.md와 STORY_GENERATION_RULES.md에 명문화
