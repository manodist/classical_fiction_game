# 🔄 이미지 교체 작업 워크플로우

이 문서는 프리퀄 스토리의 이미지를 교체할 때 따라야 할 자동화된 프로세스를 정의합니다.

> ⚠️ **중요**: 모든 날짜는 [DATE_MANAGEMENT.md](./DATE_MANAGEMENT.md)를 기준으로 합니다!

---

## 📋 작업 프로세스 (자동화)

### 0단계: 날짜 확인 (필수!)
**작업 시작 전 반드시 수행:**
1. [DATE_MANAGEMENT.md](./DATE_MANAGEMENT.md) 열기
2. "현재 날짜 정보" 섹션 확인
3. 확인한 날짜를 메모하고 작업에 사용

### 1단계: 이미지 교체 작업 수행
작품의 이미지를 시대적 배경에 맞게 교체합니다.

**원칙**:
- 작품의 시대적 배경이 절대 기준
- 페이지 제목의 키워드가 아닌 작품 자체의 시대를 기준으로 이미지 선택
- 조선시대 작품 = 100% 조선시대 이미지

### 2단계: 교체 내역 아카이빙 (자동)
교체 작업 완료 후 `/IMAGE_ARCHIVE.md`에 자동으로 기록합니다.

**기록 내용**:
```markdown
## 📋 [작품명] (YYYY-MM-DD 교체 완료)

**시대 배경**: [작품의 시대적 배경]
**주제**: [작품의 핵심 주제들]
**완성도**: ✅ 100% (N개 현대 이미지 → 전통 이미지 교체 완료)

### 🏛️ 사용된 이미지 카테고리

#### [카테고리1 이름]
```typescript
// [이미지 설명] (사용 횟수)
'[이미지 URL]'
// 사용: [사용된 장면들]
```

### 🔄 교체 완료 목록 (총 N개)
✅ **현대 이미지 → 전통 이미지 교체 완료**:
1. photo-[ID] (횟수) → [새 이미지 설명]
2. ...

### 📝 교체 이유
- **문제**: [발견된 문제점]
- **원칙 위반**: [어떤 원칙을 위반했는지]
- **해결**: [어떻게 해결했는지]
```

### 3단계: 교체 로직 규칙 업데이트 (자동)
`/IMAGE_GUIDELINES.md`와 `/STORY_GENERATION_RULES.md`에 교체 로직을 반영합니다.

**업데이트 내용**:
- 새로 발견된 원칙 위반 사례 추가
- 올바른 이미지 선택 방법 예시 추가
- 검증된 이미지 URL 추가

## 🎯 홍계월전 교체 작업 사례

### 발견된 문제
- 페이지 제목의 현대적 키워드("변화", "세대")에 현혹되어 현대 이미지 선택
- 작품의 시대적 배경(조선시대)을 무시함

### 잘못된 선택 사례
1. **"변화를 위한 용기"** 페이지
   - ❌ 선택: 현대 태권도 대회 사진
   - 🤔 이유: "변화", "용기" 키워드 → 현대적 이미지 연상
   - ✅ 수정: 조선시대 전통 무술 훈련 이미지

2. **"다음 세대를 위한 가르침"** 페이지
   - ❌ 선택: 현대 무대 사진
   - 🤔 이유: "다음 세대", "가르침" 키워드 → 현대 교육 연상
   - ✅ 수정: 조선시대 전통 교육 장면 이미지

### 확립된 원칙
**🎯 작품의 시대적 배경 > 페이지 제목의 키워드**
- 조선시대 작품은 페이지 제목이 현대적이어도 반드시 조선시대 이미지 사용
- 키워드에 현혹되지 말고 작품 전체의 시대 배경을 기준으로 선택

### 교체 통계
- **총 교체 개수**: 29개
- **교체 소요 시간**: ~30분
- **교체 카테고리**: 
  - 산 풍경 (8개)
  - 전통 궁전 (5개)
  - 자연 경로 (3개)
  - 한옥 마당 (4개)
  - 교육 이미지 (5개)
  - 기타 (4개)

## 🔍 교체 작업 시 자주하는 실수

### 1. 키워드 함정
**실수**: 페이지 제목의 현대적 키워드에 집중
**올바른 접근**: 작품 전체의 시대 배경 확인

### 2. 부분적 현대 요소 간과
**실수**: "거의 전통적인데 배경에 살짝 보이는 현대 건물"을 허용
**올바른 접근**: 조금이라도 현대 요소가 보이면 즉시 교체

### 3. 같은 이미지 과다 사용
**실수**: 한 이미지를 너무 많은 장면에 재사용
**올바른 접근**: 비슷한 카테고리 내에서도 다양한 이미지 활용

### 4. 검색 키워드 부적절
**실수**: "modern", "contemporary" 같은 단어 포함
**올바른 접근**: "traditional", "ancient", "historical" 키워드 사용
