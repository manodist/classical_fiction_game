## 📋 날짜 사용 규칙

### 1. 작업 완료 날짜
모든 문서 업데이트 시 다음 형식 사용:

```markdown
**마지막 업데이트**: YYYY-MM-DD
```

**예시**:
- ✅ `**마지막 업데이트**: 2025-11-20`
- ❌ `**마지막 업데이트**: 2024-11-20` (과거 날짜)

### 2. 작품 교체 완료 날짜
작품별 이미지 교체 작업 완료 시:

```markdown
## 📋 [작품명] (YYYY-MM-DD 교체 완료)
```

**예시**:
- ✅ `## 📋 홍계월전 (2025-11-20 교체 완료)`
- ❌ `## 📋 홍계월전 (2024-11-20 교체 완료)`

### 3. 로그 기록 날짜
변경 이력 기록 시:

```markdown
- **YYYY-MM-DD**: [작업 내용]
```

**예시**:
- ✅ `- **2025-11-20**: 홍계월전 이미지 29개 교체`
- ❌ `- **2024-11-20**: 홍계월전 이미지 29개 교체`

## 📚 참고사항

### 날짜 형식
- **전체 형식**: `YYYY-MM-DD` (예: 2025-11-20)
- **월만**: `YYYY-MM` (예: 2024-11)
- **한글 표기**: `YYYY년 MM월 DD일` (예: 2025년 11월 20일)

### 시간대
- 한국 표준시(KST) 기준
- UTC+9

### 작업일 vs 문서 업데이트일
- **작업일**: 실제 작업을 완료한 날짜 (변경 불가)
- **문서 업데이트일**: 문서를 마지막으로 수정한 날짜 (항상 최신)
