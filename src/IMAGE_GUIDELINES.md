# 📸 작품별 이미지 가이드라인

각 작품의 장면에 맞는 최적의 배경 이미지를 선택하기 위한 가이드입니다.

## ⚠️ 중요: 작품별 이미지 선택 원칙

### 🏛️ 한국 전통 작품 (조선시대 배경)

**❗ 2025-11-20 업데이트: 홍계월전 이미지 교체 작업을 통해 확립된 절대적 규칙**

**🎯 최우선 원칙: 작품의 시대적 배경이 절대 기준!**
- **키워드가 아니라 작품 시대가 기준**: "변화", "세대" 같은 단어 때문에 현대 이미지 선택 ❌
- **조선시대 작품 = 100% 조선시대 이미지**: 예외 없음!
- **원칙 위반 사례 (홍계월전)**:
  - ❌ "변화를 위한 용기" → 현대 태권도 대회 사진 (키워드에 현혹)
  - ❌ "다음 세대를 위한 가르침" → 현대 무대 사진 (키워드에 현혹)
  - ✅ 수정: 모두 조선시대 전통 무술/교육 이미지로 교체

**❌ 절대 피해야 할 현대적 요소:**
- 아파트, 빌딩 등 현대 건축물
- 전봇대, 전깃줄
- 자동차, 현대식 도로
- 콘크리트 건물
- 현대식 간판이나 표지판
- **⚠️ 산 풍경에서 보이는 현대 건물들** (특히 주의!)
- **⚠️ 한옥마을 관광 사진** (현대인이 구경하는 모습)
- **⚠️ 배경에 조금이라도 보이면 즉시 교체!**

**✅ 순수한 한국 전통 이미지만 사용:**
- 한옥, 궁궐, 전통 사찰
- 전통 정원, 한옥 마당
- 전통 기와 지붕
- 전통 문양, 단청
- 자연 풍경 (산, 강, 숲)
- **순수한 역사적 건축물만** (복원/재현 건물 제외)

## 🔍 추가 이미지 검색 팁

### Unsplash Tool 사용법
```typescript
// 예시: 바다 이미지 검색
unsplash_tool({ query: "ocean sunset" })

// 예시: 정원 이미지 검색  
unsplash_tool({ query: "garden flowers" })
```

### 검색 키워드 가이드
- **장소**: `french countryside`, `tropical island`, `old harbor`
- **자연**: `forest green`, `ocean waves`, `sunset sky`
- **건물**: `cottage house`, `castle kingdom`, `old tavern`
- **분위기**: `magical`, `mysterious`, `peaceful`
- **시대**: `old`, `vintage`, `antique`

### URL 형식
```
https://images.unsplash.com/photo-{ID}?crop=entropy&cs=tinysrgb&fit=max&fm=jpg&w=1080
```
