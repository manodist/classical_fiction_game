# 📚 디지털 독서 인터랙티브 게임 제작 규칙

## 🎯 전체 구조

### 대상 학년
- 초등학교 2-3학년
- 초등학교 4-6학년
- 중학교 1학년
- 중학교 2학년
- 중학교 3학년

### 스토리 구성
- **형태**: 프리퀄 (원작 전 이야기)
- **시점**: 2인칭 ("너")
- **최소 장면 수**: 30개 이상
- **분기**: 다양한 선택지와 결과

---

## 📋 필수 장면 구조

### 1. 시작 장면 (id: 'start')
```typescript
{
  id: 'start',
  title: '시작 제목',
  text: `2인칭 시점 텍스트`,
  bg: '배경 이미지 URL',
  choices: [
    { label: '선택지 1', to: 'scene_id', cls: 'bg-색상' },
    { label: '선택지 2', to: 'scene_id', cls: 'bg-색상' }
  ],
  prompt: '💭 질문'
}
```

**규칙**:
- 반드시 2개 이상의 선택지 필요
- 학생이 주인공이 되는 몰입형 시작

---

### 2. 중간 장면

**규칙**:
- **선택지**: 대부분의 장면에 2개 이상 제공
- **예외**: 스토리 흐름상 반드시 1개만 필요한 경우만 허용
- **선택지가 1개인 경우**: 전체 장면의 10% 미만으로 제한

**좋은 예** ✅:
```typescript
choices: [
  { label: '용감하게 도전', to: 'brave_path', cls: 'bg-red-200' },
  { label: '신중하게 준비', to: 'cautious_path', cls: 'bg-blue-200' }
]
```

**나쁜 예** ❌:
```typescript
choices: [
  { label: '계속 진행', to: 'next_scene', cls: 'bg-gray-200' }
]
```

---

### 3. 엔딩 장면 (id: 'ending') ⭐ 중요!

**필수 구조**:

```typescript
{
  id: 'ending',
  title: '프리퀄 에피소드 끝',
  text: `
[프리퀄 요약]
너는 [주인공]이었어. [주요 경험들을 나열]

[핵심 배움 정리]
- 배운 것 1
- 배운 것 2
- 배운 것 3

[본편으로의 연결]
그리고 오늘/이제...
[본편의 시작점 설명]

---

📚 [본편 스토리 소개]

[작가명]의 《작품명》에서
[본편의 핵심 내용 미리보기]

[작품의 주제나 가치 한 줄]

[THE END... 아니, THE BEGINNING!]
  `,
  bg: '적절한 배경 이미지',
  choices: [
    { label: '🔄 처음부터 다시하기', to: 'start', cls: 'bg-색상-600' }
  ],
  prompt: '💭 《작품명》 - [작품 소개 문구와 핵심 가치]'
}
```

**반드시 포함해야 할 요소**:
1. ✅ `title: '프리퀄 에피소드 끝'` (또는 유사한 표현)
2. ✅ 프리퀄 요약 + 배운 점 정리
3. ✅ 본편으로 연결되는 설명
4. ✅ 📚 책 읽기 권유 문구
5. ✅ **choices 배열에 반드시 '처음부터 다시하기' 버튼 포함**
6. ✅ prompt에 작품 소개 문구

**절대 하지 말아야 할 것** ❌:
- `choices: []` 빈 배열
- 다시 시작 버튼 없음
- 책 읽기 권유 없음

---

## 📝 텍스트 작성 규칙

### 줄바꿈
- **한 줄 줄바꿈**만 사용: `\n`
- **이중 줄바꿈 금지**: `\n\n` (과도한 공백 발생)

**좋은 예** ✅:
```typescript
text: `너는 청새치야. 바다에서 태어났어.

수많은 선택을 하며 성장했어. 강해졌어.`
```

**나쁜 예** ❌:
```typescript
text: `너는 청새치야.

바다에서 태어났어.

수많은 선택을 하며 성장했어.

강해졌어.`
```

### 시점
- **반드시 2인칭** ("너", "너는", "네가")
- 학생이 주인공이 되는 몰입감 제공

### 질문 (prompt)
- 모든 장면에 철학적/성찰적 질문 포함
- 💭 이모지로 시작
- 하부르타 학습에 적합한 사고 유도 질문

---

## 🎨 이미지 규칙

### 시대적 배경 최우선
- **절대 금지**: 현대적 요소 (자동차, 현대 건물, 영어 알파벳 등)
- **고전 작품**: 시대 배경에 맞는 이미지 (한복, 갓, 전통 건축 등)
- **자연/추상**: 시대 초월적인 자연 이미지 안전

### Unsplash 검색어
- 2-3개 키워드로 구성
- 너무 구체적이지 않게 (Unsplash 라이브러리 제한)

---

## 🔀 분기 구조

### 다양성
- 최소 3개 이상의 메인 경로
- 각 경로마다 2-3개의 서브 분기
- 최종 엔딩은 하나로 수렴

### 선택의 의미
- 모든 선택이 실제로 다른 장면으로 연결
- 의미 있는 차이 제공
- 학생의 가치관/성향에 따른 분기

---

## 📊 학년별 특성

### 초등학교 2-3학년
- 간단한 문장
- 명확한 선택지
- 모험/우정 중심

### 초등학교 4-6학년
- 조금 더 복잡한 스토리
- 도덕적 딜레마
- 고전 문학 입문

### 중학교 1-3학년
- 철학적 질문 심화
- 복잡한 심리 묘사
- 삶의 의미/가치 탐구
- 메타적 사고 유도

---

## ✅ 최종 체크리스트

제작 완료 전 반드시 확인:

- [ ] 2인칭 시점 일관성
- [ ] 30개 이상 장면
- [ ] 대부분 장면에 2개 이상 선택지
- [ ] **엔딩 페이지에 '다시하기' 버튼 있음**
- [ ] 엔딩에 책 읽기 권유 포함
- [ ] 모든 prompt에 질문 있음
- [ ] 줄바꿈 적절 (이중 줄바꿈 없음)
- [ ] 시대적 배경에 맞는 이미지
- [ ] 원작 스토리 다루지 않음 (프리퀄)
- [ ] 하부르타 학습 가능한 질문

## 📌 엔딩 페이지 예시

### 좋은 예 ✅

```typescript
{
  id: 'ending',
  title: '프리퀄 에피소드 끝',
  text: `너는 여우였어. 외로움 속에서 특별함의 의미를 배웠어.

관계는 선택이고,
시간이 특별함을 만들고,
눈에 보이지 않는 게 중요하다는 걸.

언젠가 어린왕자가 올 거야.
그때 우리는 친구가 될 거야.

---

📚 어린 왕자와 여우의 진짜 만남이 궁금해?

생텍쥐페리의 《어린왕자》에서 확인해봐!

[THE END... 아니, THE BEGINNING!]`,
  bg: 'https://images.unsplash.com/...',
  choices: [
    { label: '🔄 처음부터 다시하기', to: 'start', cls: 'bg-purple-600' }
  ],
  prompt: '💭 《어린왕자》 - 진짜 중요한 건 눈에 보이지 않아!'
}
```

### 나쁜 예 ❌

```typescript
{
  id: 'ending',
  title: '끝',
  text: `끝났어. 다시 해볼래?`,
  bg: 'https://images.unsplash.com/...',
  choices: [],  // ❌ 다시하기 버튼 없음!
  prompt: '끝'  // ❌ 작품 소개 없음!
}
```

이 게임을 통해 학생들은:

2. 등장인물과 배경에 공감
3. 철학적 질문으로 사고력 향상
4. 선택의 중요성 학습
5. 책을 읽고 싶은 동기 부여
