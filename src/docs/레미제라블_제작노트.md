# 📖 레 미제라블 프리퀄 제작 노트

## 🎯 기획 의도

### 1. **주제 선정 이유**
- **현대 이슈 연결**: 한국의 노동자 인권 문제(런던베이글, 파리바케트 SPC 등)를 19세기 프랑스 배경으로 풍자
- **공감 가능성**: 중학생들도 알바, 노동, 부당함을 이해할 수 있는 시점
- **4대 핵심 주제**: 죄, 용서, 돈, 사랑을 자연스럽게 녹임

### 2. **주인공 설정**
- **빵집 알바생** → 장발장이 빵을 훔친 바로 그 빵집에서 일하는 사람
- **2인칭 시점("너")** → 몰입감 극대화
- **평범한 사람** → 영웅이 아닌 우리와 같은 보통 사람

## 📐 선택지 설계 원칙

### 1. **무의미한 선택 없애기**
모든 선택이 의미 있도록:
```typescript
// ❌ 나쁜 예
choices: [
  { label: '그렇게 한다', to: 'next' },
  { label: '그렇게 하지 않는다', to: 'next' } // 똑같은 결과
]

// ✅ 좋은 예
choices: [
  { label: '💰 돈을 받기 위해 참는다', to: 'money_path' },
  { label: '❤️ 사장에게 항의한다', to: 'protest_path' }
]
```

### 2. **이모지로 주제 암시**
- 💰 = 돈 관련 선택
- ❤️ = 인간성 관련 선택
- 💢 = 저항/분노 관련 선택

### 3. **결과의 다양성**
- 어떤 선택을 해도 장발장은 체포됨 (원작 존중)
- 하지만 플레이어의 심경은 달라짐
- 후회, 만족, 무력감 등 다양한 감정 체험
