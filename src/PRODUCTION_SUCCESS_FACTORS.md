### 작업 규모
- **총 작업 Phase:** 8단계

- **중간 페이지 추가:** 2곳 (흐름 자연화)

## 🎯 핵심 성공 요인

### 1. **이미지 품질 관리 체계화** 🖼️

#### 문제 인식
- 초기: 사람 얼굴 사진 22곳 사용 (판타지 분위기 파괴)
- 부적절한 이미지: Apple Watch, 현대 도시 사진 등
- 테마 불일치: 같은 주제 장면에 다른 이미지 사용

#### 해결 방법
1. **IMAGE_BLACKLIST.md 생성**
   - 부적절한 이미지 URL 목록화
   - 사용처 추적 (어디서 사용되었는지 기록)
   - 교체 이력 관리 (Phase별)

2. **체계적 교체 작업**
   - Phase 1: 사람 얼굴 8곳 제거
   - Phase 2: 사람 얼굴 11곳 제거
   - Phase 3: 사람 얼굴 4곳 제거 (완전 제거 달성!)
   - Phase 4: 품질 개선 6곳
   - Phase 5: 매트릭스 스타일 1곳
   - Phase 6: 판타지 장비 1곳
   - Phase 7: 테마 정확화 5곳
   - Phase 8: 패배 장면 통일 1곳

3. **테마 일관성 확보**
   - 패배 장면 2곳 → 밤하늘의 별로 통일
   - 수도원 장면 2곳 → 실제 수도원 이미지로 통일
   - 손목 장치 2곳 → 스마트워치로 통일

4. **콘텐츠-이미지 매칭**
   - 대사: "하늘이 보인다. 별이 빛난다" → 이미지: 밤하늘의 별
   - 대사: "손목 장치를 확인한다" → 이미지: 스마트워치
   - 대사: "동굴 안으로 들어간다" → 이미지: 어두운 동굴 내부

#### 교훈
✅ **이미지 품질이 몰입감을 좌우한다**
- 사람 얼굴 사진 0개 달성 → 판타지 세계관 100% 유지
- 테마 통일 → 플레이어가 감정의 일관성을 느낌
- 콘텐츠 매칭 → "읽는 것"과 "보는 것"이 완벽히 일치

### 2. **분기 흐름의 자연스러움** 🌊

#### 문제 인식
- 논리적 비약: 쓰러진 후 → 갑자기 "문을 연다"
- 과정 생략: 남자 확인 → 바로 손목 장치
- 플레이어 혼란: "어? 이게 왜 갑자기?"

#### 해결 방법
1. **중간 페이지 추가 (2곳)**
   
   **scene-11-interrupt-check (쓰러진 남자)**
   ```
   Before: 계약 방해 → 남자를 살핀다 → 손목 장치
   After:  계약 방해 → 남자를 살핀다 → 남자 확인 페이지 → 손목 장치
   
   추가 내용:
   - 남자 맥박 확인 (살아있음)
   - 계약 효과 시작 확인
   - 남자를 바닥에 눕힘
   - 메피스토 추적 결심
   ```

   **scene-18-ultimate-aftermath (회복과 추적)**
   ```
   Before: 필살기 쓰러짐 → 일어난다 → "문을 연다"
   After:  필살기 쓰러짐 → 일어난다 → 회복 페이지 → 문을 연다
   
   추가 내용:
   - 의식 회복 (10-20분 경과)
   - 상태 확인 (HP, 마력)
   - 손목 장치로 메피스토 추적
   - 폐성 탐색 과정
   - 불빛 발견
   ```

2. **무한 루프 버그 수정**
   - scene-27-injured → scene-28 (존재하지 않음) ❌
   - scene-27-injured → ending-failure (논리적) ✅

3. **STORY_FLOW_DIAGRAM.md 생성**
   - 전체 40개 장면의 분기 구조 시각화
   - 13개 주요 분기점 명확히 표시
   - 4가지 엔딩 루트 추적 가능

#### 교훈
✅ **모든 장면 전환에는 논리적 과정이 필요하다**
- 시간 경과 명시 ("10분 후", "의식을 잃고...")
- 장소 이동 묘사 ("복도를 따라", "계단을 내려가")
- 상태 변화 설명 (HP/마력 회복, 감정 변화)
- 중간 과정 생략 금지 → 영화처럼 자연스럽게

#### 성공 사례
1. **패배 장면 통일**
   - scene-26-defeat: "하늘이 보인다. 별이 빛난다"
   - scene-29-bad: "하늘을 본다. 별이 빛난다"
   - → 둘 다 같은 이미지 (밤하늘의 별) 사용

   - 손목 장치 관련 장면들: 모두 스마트워치 이미지

- 같은 감정 = 같은 분위기 이미지
- 같은 장소 = 비슷한 스타일 이미지
- 같은 사물 = 동일한 이미지

### 5. **철학적 질문의 깊이** 💭

#### 파우스트의 질문들
```
💭 복수는 정의일까, 집착일까?
💭 늦은 정의는 무의미할까, 그래도 가치가 있을까?
💭 타인의 운명에 개입하는 것은 정의일까, 오만일까?
💭 적을 이해하는 것은 약점일까, 무기일까?
💭 전력을 다한 패배와 전력을 아낀 패배, 어느 쪽이 나을까?
💭 한 사람을 구하기 위해 자신을 희생하는 건 영웅적일까, 무모할까?
💭 실패 후 다시 일어서는 힘은 어디서 나올까?
```

#### 특징
- **양면성 제시**: A일까, B일까?
- **가치 판단 요구**: 단순한 사실이 아닌 가치관 질문
- **학생 경험 연결**: 추상적이지만 공감 가능한 수준
- **정답 없음**: 생각할 거리 제공

#### 교훈
✅ **질문이 작품의 깊이를 만든다**
- 중학교 2학년 수준: 철학적이지만 어렵지 않게
- 매 페이지마다 성찰 기회 제공
- 게임 후에도 생각이 남도록

### 6. **2인칭 시점의 일관성** 👤

#### 철저한 준수
- **100% "너" 시점 유지**
- 플레이어 = 주인공
- 모든 행동, 생각, 감정을 "너"의 것으로

#### 효과
```
Before (3인칭): "헌터가 검을 든다. 그는 분노에 떨고 있다."
After (2인칭):  "검을 든다. 분노가 치민다."

→ 2인칭이 훨씬 몰입도가 높음!
```

#### 교훈
✅ **시점을 흔들지 마라**
- 처음부터 끝까지 "너"
- 관찰자가 아닌 당사자로
- 플레이어가 주인공임을 계속 상기

### 7. **프리퀄 구조의 효과** 📖

#### 왜 원작이 아닌가?
- **원작 스포일러 방지**: 결말을 모르고 접근
- **자유로운 해석**: 세계관만 차용, 새로운 이야기
- **흥미 유발**: 원작을 읽고 싶게 만들기
- **접근성**: 원작을 몰라도 즐길 수 있음

#### 파우스트 프리퀄 구조
```
원작 파우스트: 파우스트 박사가 메피스토와 계약
프리퀘 게임: 헌터가 메피스토를 추적하며 계약자들을 구하려 함

→ 원작의 "악마와의 계약" 테마를 체험하지만, 
   원작 스토리는 건드리지 않음
```

#### 교훈
✅ **프리퀄은 원작으로 가는 다리**
- 세계관 이해 → 원작 흥미 유발
- 등장인물 공감 → 원작에서 재회
- 주제 체험 → 원작의 깊이 이해

```markdown
❌ 나쁜 매칭:
- 대사: "바다를 본다"
- 이미지: 산 풍경 (완전히 다름)

- 대사: "칼을 든다"
- 이미지: 맨손 (언급된 요소 없음)

### 검증 방법
1. 텍스트 읽으며 핵심 명사 추출
2. 이미지에 그 명사가 보이는지 확인
3. 없으면 즉시 교체

### 특별히 주의할 장면
- **감정 전환 장면**: 슬픔→기쁨 시 이미지도 어두움→밝음
- **장소 이동 장면**: 실내→실외 시 이미지도 변경
- **패배/승리 장면**: 같은 감정은 같은 분위기 이미지
```

#### Rule 5: 철학적 질문 작성법
```markdown

### 기본 형식

### 중학교 2-3학년 수준
**특징**:
- 추상적 개념 사용 가능

**특징**:
- 구체적 상황 중심
- 경험과 연결
- 감정 이입 질문
- 간단한 가치 판단

- `💭 거짓말을 해서라도 친구를 보호해야 할까?`

### 초등학교 2-3학년 수준
**특징**:
- 매우 구체적
- 일상 경험
- 단순한 질문
- 쉬운 어휘

**예시**:
- `💭 친구가 울고 있으면 어떻게 해?`
- `💭 무서울 때는 어떻게 하면 좋을까?`
- `💭 혼자 있을 때와 친구와 함께 있을 때, 어느 게 더 좋아?`

### 피해야 할 질문
❌ 너무 길고 설명적:
- `이 게임을 하고 나니, 책이 읽고 싶어졌어? 진짜 이야기에는 더 재미있는 모험이 가득해!`

❌ 정답이 명확한 질문:
- `거짓말하는 게 나쁜 걸까?` (너무 뻔함)

❌ 질문이 아닌 문장:
- `용기는 중요해!` (질문 형식 아님)
```

```markdown
- 장소 (실내/실외/자연 등)
- 날씨/시간대
- 긴장도 (평화로운 장면 vs 전투 장면)

- [ ] 같은 감정 장면들의 이미지 톤이 일관되는가?

- [ ] 기술/무기 등은 동일한 이미지를 재사용하는가?
```
