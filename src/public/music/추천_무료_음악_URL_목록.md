## 1. Pixabay Music (최고 추천!) ⭐⭐⭐

### 사이트 정보
- **URL**: https://pixabay.com/music/
- **라이선스**: CC0 (완전 무료, 출처 표시 불필요)
- **CORS**: ✅ 지원
- **품질**: 최고
- **장르**: 다양 (오케스트라, 앰비언트, 클래식 등)

### URL 구조
```
https://cdn.pixabay.com/audio/YYYY/MM/DD/audio_[hash].mp3
```

### 다크 판타지 / 오케스트라 검색 키워드
1. **"dark fantasy orchestral"** - 파우스트용 ⭐⭐⭐
2. **"epic dramatic orchestra"** - 레미제라블용 ⭐⭐
3. **"ocean ambient calm"** - 노인과 바다용 ⭐⭐⭐
4. **"fairy tale piano"** - 어린왕자용 ⭐⭐
5. **"medieval fantasy"** - 중세 배경 작품용 ⭐⭐
6. **"gothic dark"** - 고딕 분위기 ⭐⭐
7. **"cinematic epic"** - 웅장한 분위기 ⭐⭐
8. **"dramatic strings"** - 감성적 분위기 ⭐⭐

### 직접 링크 얻는 방법
```
1. Pixabay Music 방문
2. 검색어 입력 (예: "dark fantasy orchestral")
3. 음악 선택 및 페이지 열기
4. F12 (개발자 도구)
5. Network 탭 선택
6. 페이지에서 재생 버튼 클릭
7. Network에서 .mp3 파일 찾기 (Filter: "mp3")
8. 파일 우클릭 → Copy → Copy link address
9. ?filename=... 이후 부분 제거
```

### 예시 (실제 사용 가능한 음악)
```
파우스트용 추천:
- 검색: "dark fantasy orchestral"
- 인기 음악: "Dark Sanctuary", "Epic Dark", "Gothic Orchestra"

노인과 바다용 추천:
- 검색: "ocean waves ambient"
- 인기 음악: "Ocean Breeze", "Calm Sea", "Peaceful Waters"
```

## 2. Free Music Archive ⭐⭐

### 사이트 정보
- **URL**: https://freemusicarchive.org/
- **라이선스**: CC BY, CC0, Public Domain
- **CORS**: ✅ 대부분 지원
- **품질**: 좋음
- **장르**: 매우 다양

### URL 구조
```
https://files.freemusicarchive.org/storage-freemusicarchive-org/music/[path]/[filename].mp3
```

### 다운로드 링크 얻는 방법
```
1. FMA 방문
2. Genre → Soundtrack 또는 Classical 선택
3. 음악 선택
4. "Download" 버튼 우클릭 → 링크 주소 복사
```

### 필터링
- License: "Creative Commons" 또는 "Public Domain" 선택
- Genre: Soundtrack, Orchestral, Ambient 선택

## 3. Internet Archive ⭐⭐⭐

### 사이트 정보
- **URL**: https://archive.org/details/audio
- **라이선스**: Public Domain, CC0, CC BY
- **CORS**: ✅ 완벽 지원
- **품질**: 다양
- **안정성**: 최고 (영구 보관)

### URL 구조
```
https://ia[숫자].us.archive.org/[숫자]/items/[item-id]/[filename].mp3
```

### 직접 링크 얻는 방법
```
1. archive.org/details/audio 방문
2. "orchestral" 또는 "classical" 검색
3. 음악 선택
4. "DOWNLOAD OPTIONS" 섹션에서 "VBR MP3" 우클릭
5. 링크 주소 복사
```

### 추천 컬렉션
- **Classical Music Archive**
- **Orchestral Public Domain**
- **Movie Soundtracks (Public Domain)**

## 6. YouTube Audio Library ⭐⭐

### 사이트 정보
- **URL**: https://studio.youtube.com/channel/UC[your-id]/music
- **라이선스**: 무료 사용 (일부 출처 표시 필요)
- **CORS**: ❌ 직접 링크 불가

```
1. YouTube Audio Library에서 다운로드
2. /public/music/ 폴더에 저장
3. 로컬 경로 사용: musicUrl="/music/파일명.mp3"
```

**참고:** YouTube는 직접 URL 스트리밍을 지원하지 않으므로 파일 다운로드 필요

## 🎯 실전 사용 예시

### GameScreen.tsx에 URL 추가

```typescript
// 방법 1: 직접 URL 입력
{bookTitle === '파우스트' && (
  <BackgroundMusic
    musicUrl="https://cdn.pixabay.com/audio/2024/01/example.mp3"
    bookTitle={bookTitle}
  />
)}

// 방법 2: URL 맵핑
const MUSIC_URLS: Record<string, string> = {
  '파우스트': 'https://cdn.pixabay.com/audio/2024/.../dark-fantasy.mp3',
  '노인과 바다': 'https://cdn.pixabay.com/audio/2024/.../ocean-calm.mp3',
  '레미제라블': 'https://incompetech.com/music/royalty-free/mp3-royaltyfree/Brandenburg.mp3',
};

{MUSIC_URLS[bookTitle] && (
  <BackgroundMusic
    musicUrl={MUSIC_URLS[bookTitle]}
    bookTitle={bookTitle}
  />
)}
```

## 📋 URL 테스트 체크리스트

### 각 URL을 사용하기 전:

**1. 브라우저 테스트**
```
✅ URL을 새 탭에 붙여넣기
✅ 음악이 재생되는지 확인

```

**2. 개발자 도구 테스트**
```javascript
// F12 → Console
const audio = new Audio('YOUR_URL_HERE');
audio.play();
```

**3. CORS 헤더 확인**
```
F12 → Network → 파일 클릭 → Headers 탭
Response Headers에서:
Access-Control-Allow-Origin: * (✅)
```

## 📞 추가 정보

### CORS 문제 발생 시
1. **Pixabay 또는 Internet Archive 사용** (CORS 허용)
2. **로컬 파일 사용** (`/public/music/`)
3. **GitHub + jsDelivr CDN 사용**

### 저작권 확인
- ✅ CC0: 자유롭게 사용, 출처 표시 불필요
- ✅ CC BY: 자유롭게 사용, 출처 표시 필요
- ✅ Public Domain: 자유롭게 사용
- ❌ All Rights Reserved: 사용 불가

### 파일 크기 고려
- 외부 URL: 사용자 대역폭 사용
- 로컬 파일: 서버 대역폭 사용
- 권장: 3-5MB (192kbps MP3)
