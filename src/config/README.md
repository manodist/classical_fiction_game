# 🎵 배경 음악 중앙 관리 시스템

## 개요

**프리퀄로 제작된 모든 작품**의 배경 음악을 **한 파일**에서 관리합니다.  
작품별로 BGM URL, 볼륨, 활성화 여부를 쉽게 수정할 수 있습니다.

---

## 🚀 초간단 시작 (3단계!)

### 1단계: `/config/music.ts` 파일 열기

### 2단계: 작품 찾아서 URL 입력

```typescript
// 예: 노인과 바다에 BGM 추가

'노인과 바다': {
  url: 'https://cdn.pixabay.com/audio/2024/.../ocean-calm.mp3',  // 👈 URL 입력!
  volume: 0.3,
  enabled: true,  // 👈 false를 true로 변경!
  description: 'Ocean Waves Ambient - 차분한 바다 소리',
},
```

### 3단계: 저장 → 새로고침 → 완료! ✅

## 📖 작품별 구조 설명

### 각 작품은 이렇게 구성됩니다:

```typescript
'작품명': {
  // ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  url: 'https://cdn.pixabay.com/audio/.../music.mp3',
  // 👆 Pixabay URL 또는 /music/파일명.mp3

  // 👆 볼륨 (0.0 ~ 1.0)

  // 👆 관리용 메모 (선택사항)
  
  // ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  // ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   * 🎵 추천 검색 키워드:
   * - "키워드 1"
   * - "키워드 2"
   * 
   * 🎨 분위기: 설명...
   */
},
```

## 🎵 Pixabay에서 URL 얻는 방법 (5분!)

### 상세 단계

```
┌─────────────────────────────────────────────────────────┐
│ 1단계: Pixabay Music 방문                               │
├─────────────────────────────────────────────────────────┤
│ https://pixabay.com/music/                             │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│ 2단계: 작품별 추천 키워드로 검색                        │
├─────────────────────────────────────────────────────────┤
│ 예: "ocean waves ambient" (노인과 바다)                │
│ music.ts 파일의 주석에 키워드가 있습니다!              │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│ 3단계: 마음에 드는 음악 선택 및 재생                   │
├─────────────────────────────────────────────────────────┤
│ 2-3개 들어보고 가장 어울리는 음악 선택                 │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│ 4단계: 개발자 도구 열기                                 │
├─────────────────────────────────────────────────────────┤
│ Windows/Linux: F12 또는 Ctrl+Shift+I                   │
│ Mac: Cmd+Option+I                                       │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│ 5단계: Network 탭에서 mp3 URL 복사                     │
├─────────────────────────────────────────────────────────┤
│ 1. Network 탭 클릭                                      │
│ 2. Filter에 "mp3" 입력                                  │
│ 3. mp3 파일 우클릭 → Copy → Copy link address         │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│ 6단계: URL 정리                                         │
├─────────────────────────────────────────────────────────┤
│ 복사한 URL:                                             │
│ https://cdn.pixabay.com/.../audio_abc.mp3?filename=... │
│                                                         │
│ 정리한 URL (?filename=... 제거):                       │
│ https://cdn.pixabay.com/.../audio_abc.mp3             │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│ 7단계: /config/music.ts에 붙여넣기                     │
├─────────────────────────────────────────────────────────┤
│ url: 'https://cdn.pixabay.com/.../audio_abc.mp3',     │
│ enabled: true,  // 👈 활성화!                          │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│ 8단계: 저장 → 브라우저 새로고침 → 테스트! ✅          │
```

## ⚙️ 설정 옵션 상세

### URL 설정

```typescript
// ✅ 외부 URL (권장!)
url: 'https://cdn.pixabay.com/audio/2024/.../music.mp3'

// ✅ 로컬 파일
url: '/music/faust-dark-fantasy.mp3'

// ❌ 빈 문자열 (BGM 없음)
url: ''
```

---

### 볼륨 설정

```typescript
volume: 0.1  // 10% - 매우 조용 (주변 분위기)
volume: 0.2  // 20% - 조용
volume: 0.3  // 30% - 적당 (권장! ⭐)
volume: 0.4  // 40% - 적당
volume: 0.5  // 50% - 중간
volume: 0.6  // 60% - 중간
volume: 0.7  // 70% - 시끄러움 (권장 안 함)
```

**권장:** `0.3` (30%) - 독서 방해 안 함

---

### 활성화 설정

```typescript
enabled: true   // ✅ BGM 활성화 (재생 버튼 표시)
enabled: false  // ❌ BGM 비활성화 (버튼 숨김)
```

---

### 설명 (선택사항)

```typescript
description: 'Dark Fantasy Orchestral - 신비롭고 긴장감 있는 분위기'
description: 'Ocean Waves Ambient - 차분한 바다 소리'
description: ''  // 비어있어도 됨
```

**용도:** 관리자가 어떤 음악인지 기억하기 위한 메모

## 💡 관리 팁

### 효율적인 작업 순서

```
1. 작품 우선순위 정하기
   └─→ 사용 빈도가 높은 작품부터

2. 비슷한 장르끼리 묶어서 작업
   └─→ 예: 한국 고전 3개를 한 번에

3. 음악 여러 개 탭으로 열어놓고 비교
   └─→ 가장 어울리는 것 선택

4. description에 키워드 메모
   └─→ 나중에 교체할 때 참고
```

---

### URL 관리

```
✅ 외부 URL 사용 (권장!)
   - 파일 다운로드 불필요
   - 서버 공간 절약
   - 쉬운 교체
   
⚠️ 로컬 파일 사용
   - /public/music/ 폴더에 저장
   - 파일명: 작품명-bgm.mp3
   - 용량: 3-5MB (128-192kbps)
```

---

### 볼륨 조정 가이드

```
독서 집중형 작품 (철학적, 감성적)
└─→ volume: 0.2 ~ 0.3 (조용하게)
    예: 파우스트, 어린왕자

모험/액션형 작품 (역동적, 영웅담)
└─→ volume: 0.3 ~ 0.4 (적당하게)
    예: 홍길동전

감정적 작품 (슬픔, 감동)
└─→ volume: 0.3 (적당하게)
    예: 심청전, 레미제라블
```

## 💬 FAQ

### Q: 음악이 재생되지 않아요

```
A: 체크리스트:
   1. url이 정확한가?
   2. enabled: true인가?
   3. URL에 ?filename=... 없는가?
   4. 브라우저 새로고침했는가?
   5. 콘솔(F12)에 에러가 있는가?
```

---

### Q: 여러 작품에 동시에 BGM을 추가하려면?

```
A: 각 작품별로 설정하면 자동으로 적용됩니다:
   
   '파우스트': { url: '...', enabled: true },
   '노인과 바다': { url: '...', enabled: true },
   '레미제라블': { url: '...', enabled: true },
   
   GameScreen에서 자동으로 감지합니다!
```

---

### Q: BGM을 임시로 끄려면?

```
A: enabled만 변경:
   
   enabled: false,  // 👈 BGM 비활성화
   
   URL은 그대로 두고 나중에 다시 활성화 가능
```

---

### Q: 로컬 파일도 사용할 수 있나요?

```
A: 네!
   1. /public/music/ 폴더에 MP3 파일 저장
   2. url: '/music/파일명.mp3'
   3. enabled: true
```

---

### Q: 음악을 교체하려면?

```
A: URL만 변경:
   
   url: 'https://cdn.pixabay.com/.../new-music.mp3',
   
   enabled: true (그대로)
```
