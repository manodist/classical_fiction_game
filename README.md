# 📚 Classical Fiction Game

**Classical Fiction Game**은 고전 문학에 생명력을 불어넣는 대화형 AI 기반 독서 게임입니다. 역동적인 스토리텔링, 매력적인 UI, 그리고 인터랙티브한 독서 챌린지를 통해 시대를 초월한 고전 소설의 세계에 빠져보세요!
**Classical Fiction Game** is an interactive, AI-powered reading game that brings classic literature to life. Immerse yourself in the timeless worlds of classic novels with dynamic storytelling, engaging UI, and interactive reading challenges!

## ✨ 주요 기능 (Features)
- **인터랙티브 스토리텔링 (Interactive Storytelling)**: 게임화된 인터페이스를 통해 고전 소설을 새롭게 경험하세요.
- **AI 딥 인사이트 (AI-Powered Insights)**: 등장인물 분석과 스토리 확장을 AI가 역동적으로 제공합니다.
- **모던 반응형 UI (Modern & Responsive UI)**: 고급스러운 타이포그래피와 부드러운 애니메이션으로 아름답게 제작되었습니다.

## 💻 기술 스택 (Tech Stack)
- **프론트엔드 (Frontend)**: React (TypeScript) + Vite
- **스타일링 (Styling)**: Tailwind CSS + Custom Animations
- **배포 (Deployment)**: Vercel

## 🚀 빠른 시작 (Getting Started)

```bash
# Clone the repository
git clone https://github.com/manodist/classical_fiction_game.git

# Install dependencies
npm install

# Start the development server
npm run dev
```
